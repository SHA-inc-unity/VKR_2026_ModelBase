# Service Profiles

Этот каталог содержит краткие агентные профили по каждому сервису и общему каталогу.

## Как использовать

1. Определи затронутый сервис.
2. Прочитай профиль сервиса в этом каталоге.
3. После этого открой README.md и STRUCTURE.md самого сервиса.
4. После изменений обнови профиль только если изменились обязательные документы, ownership, входные точки или рабочий маршрут агента.
